# AI-3D 建模系统 - 项目说明

## 概述

AI-3D 建模系统是一个基于 OpenClaw 的智能 3D 模型生成服务。用户通过飞书发送图片或文字描述，系统自动调用多家供应商 API 生成 3D 模型。

## 系统架构

```
用户 (飞书群)
    ↓
OpenClaw Gateway (port 18789)
    ↓
AI Agent + Skill (ai-3d-modeling)
    ↓
┌─────────────────────────────────────┐
│         smart-ai-system             │
├─────────────────────────────────────┤
│ skill/         # 接收请求入口       │
│ adapters/      # 供应商适配器       │
│ db/            # 数据库管理         │
│ storage/       # TOS 存储           │
│ notifier/      # 飞书通知           │
│ poller/        # 轮询守护进程       │
└─────────────────────────────────────┘
    ↓
供应商 API (Seed3D, Hyper3D, Hitem3D)
```

## 核心模块

| 模块 | 说明 |
|------|------|
| `skill` | OpenClaw Skill 接口，接收用户请求 |
| `adapters` | 供应商适配器工厂，支持模板驱动扩展 |
| `db` | SQLite 数据库封装，管理会话/材料/任务/结果 |
| `storage` | TOS 对象存储，上传下载文件 |
| `notifier` | 飞书卡片消息通知 |
| `poller` | 轮询守护进程，获取任务结果 |

## 数据流

1. **接收请求** - 用户在飞书群发送消息
2. **意图识别** - 识别 3D 建模/取消/状态/帮助 意图
3. **创建会话** - 在数据库创建会话记录
4. **提交任务** - 为每个活跃供应商创建任务
5. **轮询结果** - Poller 定期轮询供应商 API
6. **上传存储** - 结果文件上传到 TOS
7. **发送通知** - 飞书卡片推送结果

## 供应商支持

| 供应商 | 模型 | 支持格式 |
|--------|------|---------|
| 豆包 Seed3D | doubao-seed3d | glb |
| 影眸 Hyper3D | yingmou | glb, obj, usdz, fbx, stl |
| 数美 Hitem3D | shumei | obj, glb, stl, fbx, usdz |

## 技术栈

- **语言**: Python 3.12+
- **数据库**: SQLite
- **存储**: TOS (火山引擎对象存储)
- **框架**: aiohttp, httpx
- **消息**: 飞书卡片消息
- **调度**: Poller 守护进程
