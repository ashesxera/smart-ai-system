## 二、各模块详细分析

### 2.1 adapters（适配器模块）✅ 90%

**已完成：**
- TemplateRequestBuilder：模板变量替换（支持 `${var}` 和 `${var:N}` 格式）
- TemplateResponseParser：JSONPath 风格响应解析、状态映射
- BaseAdapter：HTTP 提交/查询、认证头（bearer/api_key）
- AdapterFactory：适配器注册工厂

**未完成/问题：**
- 仅注册了 `ark_generic` 和 `generic`，无其他供应商特定适配器
- ARK API Key 从环境变量 `ARK_API_KEY` 获取，但配置文件未设置此变量
- 没有适配器级别的错误重试逻辑

### 2.2 db（数据库模块）✅ 85%

**已完成：**
- Database：连接管理、线程本地连接、表初始化
- SessionManager：会话 CRUD、状态/阶段更新
- MaterialManager：材料 CRUD、图片 URL JSON 序列化
- VendorTaskManager：任务 CRUD、轮询计数、running 状态推进
- ResultManager：结果 CRUD
- OpsLogManager：操作日志

**未完成/问题：**
- ❌ 缺少 `__main__.py`，无法通过 `python -m ai_3d_modeling.db.init` 调用
- ❌ 缺少数据库迁移脚本（`migrate` 命令在部署文档中引用但不存在）
- ✅ 已创建 `scripts/init_db.py` 作为替代初始化入口

### 2.3 notifier（通知模块）✅ 85%

**已完成：**
- ResultSummarizer：结果汇总、耗时计算、材料预览
- FeishuNotifier：飞书卡片构建、HTTP 发送

**未完成/问题：**
- ⚠️ `gateway_url` 硬编码为 `http://127.0.0.1:18789/webhook/notify`，应从配置读取
- 没有重试机制
- 没有发送失败时的降级处理

### 2.4 poller（轮询模块）✅ 90%

**已完成：**
- Poller 类：60 秒批量轮询、任务状态更新、成功/失败/超时处理
- TOS 上传触发（调用 storage）
- 汇总通知触发（调用 notifier）
- `run_poller()` 入口函数

**未完成/问题：**
- 没有进程 daemonize 包装（部署文档提到 nohup/systemd，但代码无支持）
- 没有健康检查端点
- 轮询间隔固定 60 秒，不可配置

### 2.5 skill（技能模块）⚠️ 75%

**已完成：**
- SkillHandler：意图解析（3D建模/取消/状态/帮助/未知）
- `_handle_3d_modeling`：会话创建→材料创建→提交供应商任务的完整流程
- `_submit_vendor_tasks`：批量提交活跃供应商
- `_get_active_vendors`：从 settings 表读取活跃供应商
- 帮助文本、状态查询、取消处理

**未完成/问题：**
- ❌ `_extract_images` 仅处理 `message_type == 'image'`，合并转发消息等类型未实现（代码注释 TODO）
- ❌ 缺少 OpenClaw Skill 注册配置（无 `SKILL.md` 或 skill 配置）
- ⚠️ 入口函数 `handle_event` 硬编码初始化 `Database` 和 `FeishuNotifier`，不符合依赖注入原则，难以测试
- ⚠️ 飞书消息中的图片 `image_key` 未通过飞书 API 下载为真实 URL

### 2.6 storage（存储模块）✅ 80%

**已完成：**
- StorageManager：路径验证、安全检查（`..` 路径遍历防护）
- `upload()`：tosutil cp 上传
- `download()`：tosutil cp 下载
- `generate_share_url()`：tosutil sign 生成过期链接
- `upload_result()`：下载供应商文件→上传TOS→生成链接完整流程

**未完成/问题：**
- ⚠️ 依赖 `tosutil` CLI 工具（已安装 v4.1.2），但配置文件 `~/.tosutilconfig` 还缺少 AK/SK
- 没有使用 volcengine-python-sdk 的 TOS SDK（纯 CLI 实现）
- 没有断点续传逻辑

### 2.7 utils（工具模块）✅ 95%

**已完成：**
- `generate_uuid(prefix)`：带前缀的唯一 UUID
- `get_timestamp()`：秒/毫秒时间戳
- `format_timestamp()`：人类可读时间
- `format_duration()`：时分秒格式化
- `sanitize_path()`：路径安全检查
- `build_tos_path()`：TOS 路径构建
- `extract_filename_from_url()`：URL 文件名提取

python3 scripts/init_db.py

PYTHONPATH=src python3 -m ai_3d_modeling.poller

# 5. 启动 poller（后台）
nohup env PYTHONPATH=src python3 -m ai_3d_modeling.poller > logs/poller.log 2>&1 &
