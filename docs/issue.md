# System Design v1.1 问题清单

## 严重问题

### 1. ER图显示错误
**位置**：5.2 ER关系

**问题**：ER图中出现乱码 `keeperStart1776687385544h72ir11653keeperEnd`，应该是 `<` 符号

**现状**：
```
resources (N) ───keeperStart1776687385544h72ir11653keeperEnd─── tasks (1)
```

**应显示为**：
```
resources (N) ───< resources_tasks >─── tasks (1)
```

---

### 2. 目录结构格式错误
**位置**：4.3 存储管理模块 - 目录结构

**问题**：目录结构缺少正确的层级符号

**现状**：
```
└── {api_task_id}.ext
```

**应显示为**：
```
└── {api_task_id}.ext              ← 输出文件（API返回ID）
```

---

### 3. 业务流程格式混乱
**位置**：6.1 任务提交流程、6.2 任务跟踪流程

**问题**：流程图中缺少箭头符号 `↓`，显示为普通文本

**现状**：
```
用户飞书对话

  ↓
```

**应显示为**：
```
用户飞书对话
    ↓
意图识别（判断任务类型）
    ↓
...
```

---

**位置**：第7节 参考文档
