# AI-3D 建模系统 - 使用说明

## 快速开始

### 1. 激活技能

在飞书群中发送：

```
/skill-switch ai-3d-modeling
```

### 2. 发送建模请求

发送以下任意格式的消息：

**文字描述**
```
帮我生成一个3D模型，描述：可爱的卡通人物
```

**图片建模**
```
帮我根据这张图生成3D模型
```
（附带图片）

**文字+图片**
```
根据这张图片生成一个3D模型，风格：Q版卡通
```

## 功能指令

### 生成 3D 模型

**关键词**: `3D`, `三维`, `建模`, `生成模型`, `obj`, `glb`, `stl`

**示例**:
- `生成一个3D模型`
- `帮我做一个卡通人物`
- `图片建模`

### 查询状态

**关键词**: `状态`, `进度`, `怎么样了`, `完成了吗`

**示例**:
- `我的建模任务完成了吗？`
- `查看进度`

### 取消任务

**关键词**: `取消`, `撤销`, `停止`

**示例**:
- `取消建模`
- `停止任务`

### 获取帮助

**关键词**: `帮助`, `help`, `怎么用`

**示例**:
- `怎么使用3D建模？`
- `help`

## 工作流程

```
1. 提交请求
   ↓
2. 收到回执（包含会话ID）
   ↓
3. 等待处理（1-3分钟）
   ↓
4. 收到结果通知（包含3D模型下载链接）
```

## 回执通知

提交请求后，你会收到如下回执：

```
🎨 收到3D建模请求

✅ 已提交 3 个供应商处理
📁 材料类型: 图片和文字
🆔 会话ID: sess_xxx...

⏱️ 处理时间通常需要 1-3 分钟
```

## 结果通知

任务完成后，你会收到：

```
🎉 3D建模完成！

✅ 供应商: 影眸 Hyper3D
📁 格式: glb
📦 文件: model.glb

[下载链接]
```

## 支持的 3D 格式

| 格式 | 说明 |
|------|------|
| **GLB** | 推荐，通用 3D 格式 |
| **OBJ** | 通用模型格式 |
| **STL** | 3D 打印常用 |
| **FBX** | 影视游戏格式 |
| **USDZ** | Apple 生态格式 |

## 注意事项

1. **图片要求**
   - 支持 JPG、PNG
   - 单张不超过 10MB
   - 部分供应商支持多图

2. **处理时间**
   - 通常 1-3 分钟
   - 复杂请求可能更长

3. **文件有效期**
   - 下载链接有时效
   - 请及时下载模型

4. **并发限制**
   - 同一会话同时只能有一个建模任务

## 常见问题

**Q: 任务失败怎么办？**
A: 检查是否收到失败通知，可重新发送请求。

**Q: 如何查看之前的任务？**
A: 发送"状态"查询当前会话的任务。

**Q: 支持哪些供应商？**
A: 豆包 Seed3D、影眸 Hyper3D、数美 Hitem3D。
